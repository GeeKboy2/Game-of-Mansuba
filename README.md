make -WIDTH=10 -HEIGHT=6 : Pour compiler tout les fichier avec l'option d'un monde de taille (6x10), par defaut make compile tous les fichiers avec un monde de taille (4x5).

make test : pour executer tout les fichiers qui se trouve dans le répertoire tst

make game : pour executer tout les fichier qui se trouve dans le répertoire src et jouer le jeux.

make clean : pour supprimer tout les fichiers .o qui ont servis à construire les executables.
